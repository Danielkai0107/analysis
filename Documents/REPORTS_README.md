# Accupass 活動分析報告系統

## 📊 專案說明

本專案將 Accupass 活動數據報告轉換為互動式 React 組件，提供三大類別的活動分析報告：

1. **學習類報告** (`LearningReport.tsx`) - 基於學習分析報告
2. **藝文類報告** (`ArtsReport.tsx`) - 基於藝文分析報告
3. **體驗類報告** (`ExperienceReport.tsx`) - 基於體驗分析報告

## 🚀 快速開始

### 安裝依賴

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

然後在瀏覽器中開啟 [http://localhost:3000](http://localhost:3000)


## 📈 報告功能

每個報告組件都包含以下六個分頁：

### 1. 總覽

- 活動總數統計
- 資料範圍與平均間隔
- 線上 vs 線下活動對比
- 平日 vs 假日活動分布

### 2. 活動效能

- 各分類效能排行圖表
- 詳細數據表格
- 平均觀看數與喜歡數分析

### 3. 標籤分析

- 高頻標籤 Top 9
- 標籤效能數據表格
- 熱門標籤組合

### 4. 時間趨勢

- 星期活動分布圖表
- 月份活動分布趨勢
- 平日假日統計對比

### 5. 內容吸引力

- 高互動率活動 Top 10
- 詳細排名表格
- 高觀看標題關鍵字

### 6. 市場機會

- 紅海市場分析（競爭激烈領域）
- 藍海市場分析（機會領域）
- 細分市場活動統計
- 市場洞察建議

## 🎨 技術棧

- **Next.js 16** - React 框架
- **React 19** - UI 框架
- **Recharts** - 圖表庫
- **Tailwind CSS 4** - 樣式框架
- **TypeScript** - 型別支援

## 📊 圖表類型

報告中使用了多種圖表類型來呈現數據：

- **長條圖 (BarChart)** - 活動效能、標籤分析
- **折線圖 (LineChart)** - 時間趨勢
- **圓餅圖 (PieChart)** - 平日假日分布
- **水平長條圖** - 高互動率活動排名

## 🎯 使用方式

1. 訪問首頁：http://localhost:3000
2. 點擊「📊 查看分析報告」按鈕
3. 在報告頁面選擇要查看的報告類型：
   - 學習類報告
   - 🎨 藝文類報告
   - 🌟 體驗類報告
4. 在報告中切換不同分頁查看詳細分析

## 🎨 自訂樣式

所有組件使用 Tailwind CSS 進行樣式設計，可以輕鬆自訂：

- 顏色主題在每個組件的 `COLORS` 陣列中定義
- 卡片、按鈕、表格樣式都可以在 className 中修改
- 圖表尺寸可以在 `ResponsiveContainer` 的 height 屬性中調整
